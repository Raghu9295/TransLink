Pre-requisites:
1. Clone the repo.
2. User should have working IDE (eclipse is preferable)
3. User should have latest chrome browser.
4. No restrictions to download dependencies mentioned in pom.xml


Steps to run.
1. Install TestNG plugin.
2. Go to AddFavouritesTest.class under "src/test/java", test package.
3. Right click and run as "TestNG Test".
